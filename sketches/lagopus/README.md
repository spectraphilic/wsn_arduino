# SDI-12

Install the following libraries with the Arduni library manager:
- SDI-12
- Adafruit BME280
- Adafruit SHT31
- Adafruit BusIO (required by SHT31)
- Adafruit TMP117
- Sparkfun VL53L1X
- Sparkfun MLX90614

Sensors:
- 0x44 SHT31
- 0x48 TMP117
- 0x77 BME280
- 0x29 VL53L1X
- 0x5A MLX90614


# Pins

Data:
- Waspomote     : DIGITAL8
- Feather-M0    : D5
